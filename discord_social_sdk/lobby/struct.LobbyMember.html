Struct LobbyMember

pub struct LobbyMember { /* private fields */ }
Expand description

A single member of a Lobby.

Members are not removed automatically when they close the game or temporarily disconnect, so a member may exist while connected is false.

pub fn can_link_lobby(&self) -> bool

Whether this user is allowed to link a channel to the lobby.

Under the hood this checks the CanLinkLobby member flag, which can only be set through the server API’s add_lobby_member. It exists so a game can restrict channel linking to, say, the clan or guild leader.

Developer-supplied metadata for this member.
